## 五、信息获取原则

### 5.1 五条核心原则

| 原则 | 说明 |
|------|------|
| **一手优先** | 优先直接阅读官方文档和原始论文 |
| **时间敏感** | 这个领域变化极快，6 个月前的信息可能已过时 |
| **警惕营销** | 商业公司的宣传可能夸大产品能力 |
| **交叉验证** | 关键信息应从多个来源验证 |
| **实践导向** | 理论应与实际使用经验交叉验证 |

### 5.2 禁止使用的来源

| 来源类型 | 原因 |
|----------|------|
| 新闻媒体报道 | 通常滞后 6-12 个月且易失真 |
| 营销号和自媒体"震惊体" | 缺乏技术深度，过度简化 |
| 未经验证的社交媒体转发 | 可能传播错误信息 |

### 5.3 需谨慎对待的来源

| 来源类型 | 注意事项 |
|----------|----------|
| 商业公司的性能声明 | Agno 等公司声称的性能数据未经独立验证 |
| 预印本论文 | arXiv 论文未经同行评审 |
| 社区讨论 | 需判断发言者的专业背景 |
